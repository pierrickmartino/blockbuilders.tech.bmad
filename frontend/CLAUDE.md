# frontend/CLAUDE.md

## Commands
```bash
cd frontend
npm run dev          # dev server → localhost:3000
npm run build        # production build
npm run lint         # ESLint (must pass before commit)
npx tsc --noEmit     # TypeScript type-check (must pass before commit)
npm run storybook    # Storybook → localhost:6006
```

## Design Context

Design/brand context (audience, voice, aesthetic direction, anti-references, principles) lives in `.impeccable.md` at the project root. 
Read it before any UI or visual work.
